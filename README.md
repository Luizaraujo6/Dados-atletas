# Dados-atletas

Este projeto é uma aplicação em JavaScript que gerencia dados de atletas. Ele cria uma estrutura organizada capaz de calcular IMC, médias de notas e definir categorias de competição automaticamente.

# Sobre o Projeto

O objetivo é criar um sistema que receba o nome, idade, peso, altura e notas de um atleta e retorne um relatório completo. Diferente de usar variáveis soltas, utilizando uma classe para agrupar todas as informações.
